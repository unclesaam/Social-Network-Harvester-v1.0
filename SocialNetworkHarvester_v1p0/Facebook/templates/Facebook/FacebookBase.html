Guide d'utilisation {% include "tool/tables/tableMenuItems.html" with name="openCloseIcon" %}
La page d'accueil Faceboom vous permet d'ajouter et de gérer vos sources de contenu Facebook. Avant de continuer, assurez-vous d'avoir connecté votre compte SNH à un compte Facebook d'utilisateur. Pour ce faire, allez à votre page de paramètres et connectez-vous à Facebook dans la section Facebook
  • Ajouter une nouvelle page: Afin d'ajouter une nouvelle page publique, il suffit d'ouvrir la section "Page publiques collectées" et de cliquer sur "ajouter". Dans le champ approprié, entrer l'url de la page désiré et cliquer sur Soumettre. Puisque votre compte SNH ne collecte pas les status en temps réel, vous devez attendre un certain temps avant d'avoir accès à ceux-ci.
  • Visualiser les status: Pour visualiser les status postés sur une page Facebook, ouvrez la section Pages publiques collectées et la section Status collectés. Sélectionnez une ligne dans le premier tableau et voyez apparaitre les status dynamiquement dans le tableau des status. Il est possible de sélectionner plusieurs sources de status à la fois.
  • Télécharger les données sous la forme de tableau: Pour télécharger les données en format .csv, sélectionnez d'abord les ligne du tableau que vous désirez obtenir. (Toute les sélectionner au besoin) et cliquez sur "Télécharger la sélection". Sélectionnez ensuite les champs désirés pour chaque ligne (Seulement ceux qui vous sont nécéssaires!). Finalement, cliquez sur "Télécharger".
